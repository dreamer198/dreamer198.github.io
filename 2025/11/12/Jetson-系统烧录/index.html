Jetson Orin NX 系统烧录全流程（SDK Manager）

发表于2025-11-12|更新于2025-11-12|技术分享
|总字数:649|阅读时长:2分钟|浏览量:

适用机型：Jetson Orin NX（其它 Jetson 机型流程类似）。使用 NVIDIA 官方 Jetson SDK Manager 完成系统镜像烧录与驱动安装。

一、准备工作

  • 一台运行 Ubuntu 20.04/22.04 的电脑（建议 20.04/22.04，x86_64）
  • USB Type‑C 数据线（支持数据传输）
  • Jetson SDK Manager（NVIDIA 官方工具）
  • Jetson Orin NX 开发板、12V 供电与显示器（HDMI/DP）

软件下载：

  • SDK Manager: https://developer.nvidia.com/sdk-manager
  • 需登录 NVIDIA 账号（没有先注册）

二、安装 SDK Manager

  1. 下载 .deb 包后双击安装，或执行：
  1. 启动并登录（Ubuntu 菜单搜索 “SDK Manager”）。

三、进入恢复模式（Recovery）

  1. 用 USB‑C 线连接 PC 与开发板短边的 Type‑C 口（USB 口旁）
  2. 断电 → 按住“REC/Recovery”键 → 上电 → 2 秒后松手
  3. 此时风扇通常不转，PC 端可通过 lsusb 看到 NVIDIA 设备（可选检查）

常见位置提示：

  • REC 键与电源键位置请参考官方丝印；若不确定，尝试多次按键顺序（先按住 REC，再上电）

四、SDK Manager 配置与烧录

  1. Select Product Category: Jetson
  2. Hardware Configuration: 选择 Jetson Orin NX
  3. Target Operating System: Linux for Tegra
  4. Target Version: 选择推荐/最新 JetPack（与板卡兼容）
  5. 选择 Host/Target 组件（默认即可）
  6. 点击 Continue，按引导下载镜像与依赖 → 开始 Flash

烧录阶段注意：

  • 全程保持 USB 连接稳定，避免移动触碰
  • 若网络慢，建议先开代理或稳定网络
  • 出现权限提示时，按指引使用 sudo 重试

五、首次开机向导

烧录完成后，断开 USB，接入显示器与键鼠，重启进入系统：

  • 设置语言、时区、用户名/密码
  • 进入桌面后建议先运行：

六、常见问题排查（FAQ）

  1. PC 不识别设备（lsusb 看不到 NVIDIA）：
  • 检查是否严格按“按住 REC → 上电”顺序
  • 更换 USB‑C 线（必须支持数据）与 USB 口
  • 更换 PC 或操作系统版本（建议 Ubuntu 20.04/22.04）
  1. SDK Manager 卡在下载/校验：
  • 切换网络或配置代理后重试
  • 退出后重新登录 SDK Manager，会复用已下载数据
  1. 烧录成功但无法进桌面：
  • 尝试重启；检查电源/显示器连接
  • 若持续异常，可重新进入 Recovery 再次烧录
